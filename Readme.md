# Cat vs Dog Image Classifier

This project uses Deep Learning (CNN) to classify images of cats and dogs.

## Technologies Used
- Python
- TensorFlow
- Keras
- CNN
- Matplotlib

---

## Setup in VS Code

### Step 1
Open the project folder in VS Code

### Step 2
Create virtual environment

python -m venv venv

### Step 3
Activate virtual environment

Windows:
venv\Scripts\activate

### Step 4
Install dependencies

pip install -r requirements.txt

### Step 5
Run the project

python main.py

---

## Dataset

Download dataset from Kaggle:
Dogs vs Cats Dataset

Place images inside:

dataset/train/cats
dataset/train/dogs
dataset/test/cats
dataset/test/dogs
